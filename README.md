# Snapcart

Snapcart es una aplicación que permite administrar el carrito de compras de un usuario a traves de fotos que se le captura al producto gestionando la lista de compras del usuario.

## Funcionalidades principales

- Captura de fotos de productos para chekear o agredar en el carrito de compras.
- Calcular el total de la compra.
- Notificaciones de productos caros respecto a compras anteriores, un producto no se encuentra en la lista o si ya se encuentra, etc.
